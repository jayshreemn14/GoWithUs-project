# GoWithUs - Travel Booking Web App

A travel booking web application built using Flask and MySQL.

## Features
- Browse 50+ travel destinations
- View detailed destination pages
- Book trips with check-in & check-out
- Automatic total price calculation
- View booking history
- Clean UI using HTML & CSS

## Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS
- SQLAlchemy

## How to Run
1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open browser:
   http://127.0.0.1:5000
